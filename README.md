# Awesome-books

This is a simple book application that supportsadding books, removing books and keeping the books in local storage


## Built With

- HTML.
- CSS.
- JavaScript.


## Getting Started

- To run a local copy of this project follow these simple steps.

### Prerequisites

- A modern web browser, preferably Chrome or Firefox

### Cloning the repository

- Create a directory.
- Open a terminal in this directory and run this command: `git clone git@github.com:Yilikal250/Awesome-book.git`.
- Open the index.html file with a browser application
- The website will load
